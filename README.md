# Daily Expense Tracker 

A clean and modern web app to track daily expenses, view summaries,
set budgets, and visualize spending patterns.

### Features:
- Add / Edit / Delete expenses
- Category chart (doughnut)
- Monthly budget progress
- Filters (Today / This Month / All Time)
- Search + sorting
- Login / signup (local storage)
- Toast notifications
- Clean UI with glassmorphism
